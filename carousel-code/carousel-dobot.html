  • Tool Coordinate System

    This tutorial walks you through the process of creating a Tool Coordinate System for your Dobot CR-Series Cobot using DobotSCStudio. It also demonstrates how to use your new Tool Coordinate System in jogging your robot.

  • Pick and Place

    This tutorial will guide you through the process of setting up a basic pick and place routine with your Dobot CR Series Cobot, DobotSCStudio, and a DH-Robotics AG-Series gripper.
